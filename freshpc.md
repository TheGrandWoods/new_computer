# Fresh Windows 11 install checklist for your PC

## Install first

-   [Windows 11 ISO / Media Creation Tool](https://www.microsoft.com/software-download/windows11) — create the installer USB and do the clean install. [web:24]
-   [ASUS ROG STRIX B450-F GAMING II support page](https://www.asus.com/supportonly/rog%20strix%20b450-f%20gaming/helpdesk_download/) — grab the latest motherboard BIOS, chipset, LAN, audio, and utility downloads for your board. [web:1][web:22]
-   [NVIDIA GeForce drivers / NVIDIA App](https://www.nvidia.com/en-us/geforce/drivers/) — install the latest RTX 3070 Game Ready driver, or Studio Driver if you prefer creator-focused stability. [web:8][web:11]

## Motherboard and platform

-   [AMD Chipset Drivers](https://www.amd.com/en/support/download/drivers.html) — install this after Windows; it is the main platform driver package for your Ryzen 5 3600 / B450 system. [web:9][web:12]
-   [ASUS Armoury Crate](https://rog.asus.com/motherboards/rog-strix/rog-strix-b450-f-gaming-ii-model/) — optional, useful if you want ASUS-controlled RGB, board utilities, or driver discovery from one app. [web:19][web:22]

## Useful utilities

-   [Microsoft PowerToys](https://learn.microsoft.com/en-us/windows/powertoys/) — highly recommended for FancyZones, PowerRename, Run, and other productivity tools. [web:23]
-   [7-Zip](https://www.7-zip.org/) — essential archive utility for drivers, mods, and general file handling. [web:4][web:5]
-   [Microsoft Visual C++ Redistributables](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist) — common runtime package many games and apps need. [web:18]
-   [Notepad++](https://notepad-plus-plus.org/) — handy for config files, JSON, scripts, and modding work. [web:18]

## Optional but useful

-   [AMD Ryzen Master](https://www.amd.com/en/products/software/ryzen-master.html) — optional monitoring/tuning tool, not required for normal use. [web:15][web:26]
-   [CPU-Z](https://www.cpuid.com/softwares/cpu-z.html) — useful for quick hardware verification after the reinstall.
-   [HWiNFO](https://www.hwinfo.com/download/) — useful for checking temps, clocks, RAM speed, and sensors.
-   [CrystalDiskInfo](https://crystalmark.info/en/software/crystaldiskinfo/) — useful for checking SSD health on both drives.
-   [Malwarebytes](https://www.malwarebytes.com/) — optional second-opinion scanner if you want a cleanup tool.
-   [Discord](https://discord.com/download) and your game launchers — install only if you use them regularly.

## What I’d skip

-   Extra motherboard “driver packs” from random sites. Use ASUS, AMD, NVIDIA, and Microsoft only for the core setup. [web:1][web:8][web:9][web:24]
-   CPU overclocking tools unless you actually want to tune performance. Your Ryzen 5 3600 and B450 board are perfectly fine at stock with updated chipset drivers. [web:12][web:19]

## Best order

1. Install Windows 11.
2. Run Windows Update once.
3. Install AMD chipset drivers.
4. Install NVIDIA GPU drivers.
5. Install ASUS motherboard utilities only if you need them.
6. Add PowerToys, 7-Zip, and the rest of your daily-use apps.
7.
