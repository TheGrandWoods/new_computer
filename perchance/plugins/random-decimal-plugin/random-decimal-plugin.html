Random Decimal Plugin

This plugin allows you to generate a random decimal number between two numbers that you provide. The built-in \{1-10\} type syntax is fine for most cases, but what if you wanted to choose a random number between two randomly chosen numbers? Or between two user-input numbers? That's where this plugin would come in handy. To use this plugin, you'll first need to import it by putting this code in your Perchance code panel: 

randomDecimal = \{import:random-decimal-plugin\}

And here's a simple example of how to use it:

output
	Between the numbers \[s = small.selectOne\] and \[b = big.selectOne\], there is the decimal number \[randomDecimal(s, b)\].

small = \{10-20\}.{1-9}
big = \{50-100\}

and that would output this:

Between the numbers [s = small.selectOne] and [b = big.selectOne], there is the number [$output(s, b)].

Note that you can do all the normal stuff with this plugin, like assigning the resulting number to a variable like this: 

output
	The distance is \[n = randomDecimal(10, 20)\], which is \[50-n\] less than 50.

The winning number is [n = $output(10, 20)], which is [50-n] less than 50.

If you want to only display a certain number decimal places, then use .toFixed(N), where N is the number of decimal places, like this: 

\[randomDecimal(10, 20).toFixed(3)\] // random decimal between 10 and 20 with 3 decimal places
\[randomDecimal(-10, 10).toFixed(2)\] // random decimal between -10 and 10 with 2 decimal places

Note that .toFixed(N) converts the number into text, and you can't do math (like addition, subtraction, multiplication, etc.) with text, so make sure you do calculations with the number and then use .toFixed(N) to convert it into text just before outputting it. Here's an example: 

output
	The distance is \[n = randomDecimal(10, 20)\], which is \[(50-n).toFixed(2)\] less than 50.

Notice that we need to put brackets around 50-n so that the toFixed function is applied to the result of 50-n, rather than to n before it has been subtracted from 50.

And remember that you can name your plugin variable whatever you like, so if writing randomDecimal a lot becomes tiresome, you can import it like this: 

rand = \{import:random-decimal-plugin\}

And now you can just write \[rand(from, to)\].
