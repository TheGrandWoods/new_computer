Download Button Plugin

If you'd like to allow people to download your generator so they can use and edit it when they're offline, simply put this in your HTML panel: 

downloadButton = \{import:download-button-plugin\}

And then put this in your HTML panel:

\[downloadButton()\]

And you'll get a button like this:

[$output()]

Once you've downloaded a generator, you can edit it by doing the following: (1) Open the downloaded HTML file in your browser, (2) add #edit to the end of the url, (3) press enter, (4) refresh the page. Here's a demonstration video. So if there's a "nuclear winter" scenario where perchance.org is offline, you'll still be able to use and edit your generators, and you can host them on a free service like Netlify, Cloudflare Pages, or sites like Squarespace/Wix.

If you'd like to customize the text or style of the button, put this in your HTML panel instead: 

\[downloadButton("My button text", "background:green; color:white; border:none;")\]

And you'll get something like this:

[$output("My button text", "background:green; color:white; border:none;")]

The background:green; color:white; ... stuff is CSS.



Notes:

  • You can download your generator by clicking the link in your settings menu. This plugin is just for if you want to make it super easy for other people to download your generator.
  • As you can see if you open the editor on this page, this plugin simply creates a download link to https://perchance.org/api/downloadGenerator?generatorName=your-generator-name and note that you can also add &listsOnly=true to the end of that URL to only gets the lists text. E.g. https://perchance.org/api/downloadGenerator?generatorName=noun&listsOnly=true
