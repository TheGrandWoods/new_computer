Lockable List Plugin

Put this in your Perchance code panel:

lockableList = \{import:lockable-list-plugin\}


Now if you write this:

That \[lockableList(animal)\] is \[lockableList(adjective)\].

You'll get something like this (see example):

That [$output(animal)] is [$output(adjective)].

You can click on the lock to "lock" the list so that that specific list always returns that specific value.


By default the "lock" button is shown next to the list output, but you can put the lock button for a particular list anywhere you want:

\[lockableList(adjective, "button")\]

That'll output the lock button for the adjectives list (see example).


Notes:

  • Note: You may want to check out the locker-plugin, which is a more powerful (though slightly more complicated) version of this plugin. This plugin (lockable-list) doesn't work if your list items have curly/square blocks in them. It only works if your list items are just "plain old text". The locker-plugin has a bunch of other advantages too, including the ability to change the lock/unlock icons.
  • It combines nicely with the makeTable plugin: example
  • One potential weak-spot of this plugin is that it locks lists, so if you want to use a particular list multiple times and have the ability to lock them independently, then you'll need to make "duplicates" of that list - see this example (or use the locker-plugin instead).
