📅 Date Plugin 🕒

This plugin makes it easy to work with dates and times. For example, if you wanted to generate today or tomorrow's date in a specific format, then this makes it easy. To use this plugin, add this code to the top of your Perchance code panel: 

date = \{import:date-plugin\}

And now your can write code like this:

output
  Tomorrow 's date is \[date().add(1, "days").format("YYYY-MM-DD")\].

And that would output:

Tomorrow's date is [$output().add(1, "days").format("YYYY-MM-DD")].

You can also subtract days:

output
  Six days ago was \[date().subtract(6, "days").format("MMMM Do")\].

And that would output:

Six days ago was [$output().subtract(6, "days").format("MMMM Do")].

There are lots of different formatting options. Here's a list of examples: 

\[date().format("MMMM Do, YYYY")\]            =    [$output().format("MMMM Do, YYYY")]
\[date().format("dddd")\]                     =    [$output().format("dddd")]
\[date().format("MM Do YY")\]                 =    [$output().format("MM Do YY")]
\[date().format("YYYY")\]                     =    [$output().format("YYYY")]
\[date().add(3, "years").format("YYYY")\]     =    [$output().add(3, "years").format("YYYY")]
\[date().format("MMMM Do YYYY, h:mm:ss a")\]  =    [$output().format("MMMM Do YYYY, h:mm:ss a")]
\[date().format("dddd \[the\] Do")\]            =    [$output().format("dddd [the] Do")]
\[date().format("ddd, hA")\]                  =    [$output().format("ddd, hA")]
\[date().format("YYYY MM DD")\]               =    [$output().format("YYYY MM DD")]
\[date().format("\[Today is\] dddd")\]          =    [$output().format("[Today is] dddd")]
\[date().format("DDD")\]                      =    [$output().format("DDD")] (day of year)
\[date().format("DDDo")\]                     =    [$output().format("DDDo")] (day of year)
\[date().format("DDD/YYYY")\]                 =    [$output().format("DDD/YYYY")]
\[date().format("W / YYYY")\]                 =    [$output().format("W / YYYY")] (week / year)
\[date().format("Wo \[week of\] YYYY")\]        =    [$output().format("Wo [week of] YYYY")]
\[date().format("d")\]                        =    [$output().format("d")]  (day of week)
\[date().format("do")\]                       =    [$output().format("do")] (day of week)
\[date().format("dd")\]                       =    [$output().format("dd")]
\[date().format("ddd")\]                      =    [$output().format("ddd")]
\[date().format("dddd")\]                     =    [$output().format("dddd")]
\[date().format("M")\]                        =    [$output().format("M")]  (month of year)
\[date().format("Mo")\]                       =    [$output().format("Mo")] (month of year)
\[date().format("MM")\]                       =    [$output().format("MM")] (month of year)

\[date().add(2, "months").format("MMMM")\]    =    [$output().add(2, "months").format("MMMM")]
\[date().format("D")\]                        =    [$output().format("D")]  (day of month)
\[date().format("Do")\]                       =    [$output().format("Do")] (day of month)
\[date().format("DD")\]                       =    [$output().format("DD")] (day of month)

You can also use .calendar() to display a date in a format that makes sense given how far it is away from the current date: 

\[date().add(2, "hours").calendar()\]        =    [$output().add(2, "hours").calendar()]
\[date().add(24, "days").calendar()\]        =    [$output().add(24, "hours").calendar()]
\[date().add(4, "days").calendar()\]         =    [$output().add(4, "days").calendar()]
\[date().add(6, "days").calendar()\]         =    [$output().add(6, "days").calendar()]
\[date().add(7, "days").calendar()\]         =    [$output().add(7, "days").calendar()]
\[date().add(20, "days").calendar()\]        =    [$output().add(20, "days").calendar()]
\[date().add(20, "months").calendar()\]      =    [$output().add(20, "months").calendar()]
\[date().subtract(20, "hours").calendar()\]  =    [$output().subtract(20, "hours").calendar()]
\[date().subtract(2, "days").calendar()\]    =    [$output().subtract(2, "days").calendar()]

And you can also use .fromNow() to display a date as an amount of time starting from "now": 

\[date().add(2, "hours").fromNow()\]        =    [$output().add(2, "hours").fromNow()]
\[date().add(24, "days").fromNow()\]        =    [$output().add(24, "hours").fromNow()]
\[date().add(4, "days").fromNow()\]         =    [$output().add(4, "days").fromNow()]
\[date().add(6, "days").fromNow()\]         =    [$output().add(6, "days").fromNow()]
\[date().add(7, "days").fromNow()\]         =    [$output().add(7, "days").fromNow()]
\[date().add(20, "days").fromNow()\]        =    [$output().add(20, "days").fromNow()]
\[date().add(20, "months").fromNow()\]      =    [$output().add(20, "months").fromNow()]
\[date().subtract(20, "hours").fromNow()\]  =    [$output().subtract(20, "hours").fromNow()]
\[date().subtract(2, "days").fromNow()\]    =    [$output().subtract(2, "days").fromNow()]
