Roman Numerals Plugin

Put this in your Perchance code panel:

roman = \{import:roman-numerals-plugin-plugin\}


Now you can convert a number into its roman numeral form:

\[roman(18)\]

That would generate XVIII.


You can of course use it with variables and other plugins:

\[roman(num)\]

\[roman( dice("2d6") )\]

\[roman( character.hitpoints )\]


Test out this plugin by typing in a number:

\[roman()\]

[$output(userval.value)]
