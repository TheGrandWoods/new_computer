[$output(menu)]

navbar plugin






Here's an example generator showing how you can use this plugin. It allows you to easily add a customisable "nav bar" to your generator like you see above. Since Perchance doesn't have "user profiles", I figured something like this would be useful for people to consolidate their generators. Here's how to use it:

Put this in your Perchance code panel:

navbar = \{import:navbar-plugin\}

menu
  // menu options:
	title = NavBar Plugin
	titleUrl = https://example.com
	theme = \{dark|light|red|indigo|cyan|green|amber|brown|bluegrey\}
	alignment = right
	placement = static
	// menu items:
	items
		Item 1 | https://example.com
		Item 2 | https://example.com
			Sub-item 1 | https://example.com
			Sub-item 2 | https://example.com
		Item 3 | https://example.com

And put this in your HTML panel:

\[navbar(menu)\]

If you had quite a few generators, and you wanted the same nav-bar on all of them, then you'd probably want to create a seperate "menu" generator named e.g. "my-cool-menu" (you can change generator name/url in settings) and its code would look like this: 

$output = \[menu\]

menu
	title = My Cool Menu
	titleUrl = https://example.com
	...

and then import it into all your other generators like this:

navbar = \{import:navbar-plugin\}
menu = \{import:my-cool-menu\}
// and then put this in your HTML editor like normal:
\[navbar(menu)\]

That way, you don't have to copy-and-paste your menu hierarchy to each of your generators, and when you want to edit it, you just need to change it once, and it will be updated across all your generators.

You can specify your own custom theme colors like so: 

theme = blue orange red
theme = #887722 #aa8854 #ea71ff // <-- hex color notation
theme = #887722 #aa8854 pink // <-- you can mix notations
// the format is:
theme = backgroundColor hoverColor textColor

Notes:

  • It doesn't support more than 2 "levels". So you can have item>subitem, but not item>subitem>subsubitem.
  • The allowable values for theme are: dark, light, red, indigo, cyan, green, amber, brown, bluegrey.
  • The allowable values for alignment are: left and right.
  • The allowable values for placement are: default, static, fixed, fixed-bottom. The "fixed" ones will scroll with the page.
  • Try resizing the page - the menu folds away nicely into a icon if the screen is too narrow (e.g. on mobiles).
