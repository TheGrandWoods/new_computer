Tornado Plugin 🌪

Extremely useful plugin that makes all your page elements rotate. Simply put this in your Perchance code panel: 

tornado = \{import:tornado-plugin\}

And then put this in your HTML panel:

\[tornado(1)\]

You can change the 1 value to make it faster or slower (default is 0.5).

To make the plugin activate with a button (like above), use this HTML code instead: 

<button onclick="tornado(1)">my button</button>


Notes:

  • This is a very important plugin
  • Set the speed extremely small (e.g. \[tornado(0.0001)\]) to create confusion
  • Check out more plugins at perchance.org/plugins
