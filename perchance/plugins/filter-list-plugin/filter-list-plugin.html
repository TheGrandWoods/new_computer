Filter List Plugin

Put this in your Perchance code panel:

filterList = \{import:filter-list-plugin\}

Now if you have a list like this:

animal
  goat
    height = 1
    isMammal = true
    coat = fur
  ant
    height = 0.001
    isMammal = false
    coat = carapace
  horse
    height = 1.8
    isMammal = true
    coat = fur
  frog
    height = 0.02
    isMammal = false
    coat = skin
  ...

Then you can filter items by their properties:

\[filterList(animal, item => item.height > 1.2)\] // height greater than 1.2
\[filterList(animal, item => item.height == 1)\] // height is exactly 1
\[filterList(animal, item => item.coat == "skin")\] // notice the quotation marks for non-numbers

You just need to change animal to the name of your list, and the item.height > 1.2 parts to suit your purpose.

If that part evaluates to false (or zero, or 'undefined'), then it is filtered out.

You can do multiple conditions like item.mass > 2 && item.shape == "round". For example, using the animal example above we could grab all animals with skin that are uner 0.3 height: 

\[filterList(animal, item => item.coat == "skin" && item.height < 0.3)\]

&& (double ampersand) means AND, || means OR

Use brackets to group stuff: (item.hasBones && item.isFurry) || item.isMammal


You can use the resulting list just like you'd use any other list - i.e. you can reference it with a variable and then selectOne item from it, use .joinItems(...), and so on: 

output
  \[bigAnimals = filterList(animal, item => item.height > 1.2), ""\]Here are three big animals: \[bigAnimals.selectUnique(3).joinItems(" ")\]

You can actually shorten item to i (or any other letter): 

\[filterList(animal, i => i.isMammal)\]

It's just a "placeholder" for the item being "checked" by the filter.
