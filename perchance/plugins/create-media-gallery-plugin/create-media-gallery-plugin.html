Create Media Gallery Plugin

Import it into your lists editor like this:

createMediaGallery = \{import:create-media-gallery-plugin\}

Use it like this:

// Create the gallery:
let gallery = createMediaGallery();

// Show the gallery:
gallery.show()

// Add content to the gallery:
const blob = await fetch("https://example.com/foo.jpeg").then(r=>r.blob());
await gallery.add(blob); // adds to 'active' folder by default

// Add to a specific folder:
await gallery.add(blob, \{folder:"test1"\});

// Add custom data/info along with the entry:
await gallery.add(blob, \{info:\{foo:"123"\}\}); // only supports string/text values for now

// Get active folder (note that it's remembered between page refreshes):
let activeFolderName = gallery.getActiveFolderName();

Note that you don't need to put 'await' before 'gallery.add' - it'll still work if you don't.




Feel free to import this plugin - the API (i.e. show, add, getActiveFolderName) is stable, but the gallery interface may be adjusted in the future, so you may want to fork it if you don't want any interface changes. Also, the backup file format may change, but it'll be backwards compatible - i.e. I'll make sure loading older backups still works fine in newer versions of this plugin.
