Make Table Plugin

Put this in your Perchance code panel:

makeTable = \{import:make-table-plugin\}

Now create a list that looks like this:

myTable
	cell | cell | cell | cell
	cell | cell | cell | cell
	cell | cell | cell | cell
	cell | cell | cell | cell

Now include this in your HTML panel (bottom-right):

\[makeTable(myTable)\]

And it'll output this:

[$output(myTable)]

You can optionally change the widths and alignments of the columns like this: 

myTable
	columnWidths = 100|50|40|200
	columnAlignments = right|left|center|right
	cell | cell | cell | cell
	cell | cell | cell | cell
	cell | cell | cell | cell
	...
[$output(myTable2)]

You can add custom CSS styling to the table, rows, and cells as follows: 

myTable
	columnWidths = 100|50|40
	tableStyle = table-layout:fixed;
	cellStyle = box-sizing:border-box; border: 1px dashed red;
	rowStyle = background:lightblue
	cell | cell | cell
	cell | cell | cell
	cell | cell | cell
	...
[$output(myTable3)]

The tableStyle = table-layout:fixed; and cellStyle = box-sizing:border-box; properties makes the columnWidths "strict". If you don't add them, the column widths will "flex" if the content is too big.


Notes:

  • You can also change just the overall width of the table by setting the width property - e.g. width = 800
  • You can make a bold header row by just wrapping the cell text with <b> tags like this: <b>my cell text</b> (this is just regular HTML)
