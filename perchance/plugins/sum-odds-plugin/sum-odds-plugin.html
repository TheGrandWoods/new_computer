sumOdds Plugin

This plugin allows you to easily sum up all the odds of all the items in a particular list. It uses the item.getOdds property behind the scenes. To use it, paste this in your generator's code editor: 

sumOdds = \{import:sum-odds-plugin\}

Sometimes you want to "merge" two lists, but preserve their odds as if you've just copy and pasted all the items into one big list. In that case, you can do something like this: 

animal
	\[mammal\]^4
	\[bird\]^3

mammal
	koala
	mouse
	rabbit
	dog

bird
  chicken
  parrot
	crow

But then you have to keep updating the ^4 and ^3 every time you add/remove items from the mammal or bird lists. So you could improve it by doing this: 

animal
	\[mammal\]^\[mammal.getLength\]
	\[bird\]^\[bird.getLength\]

mammal
	koala
	mouse
	rabbit
	dog

bird
  chicken
  parrot
	crow

But that's still not super ideal because what if (for some reason) you wanted to make the "crow" item 20 times more likely, like this? 

...

bird
  chicken
  parrot
	crow^20

The ^\[bird.getLength\] approach doesn't work now because the length of the bird list is still 3, and so ^\[bird.getLength\] resolves to ^3 when we really want it to resolve to ^22 (the sum of all the odds in the bird list). Hence this plugin: 

animal
	\[mammal\]^\[sumOdds(mammal)\]
	\[bird\]^\[sumOdds(bird)\]

mammal
	koala
	mouse
	rabbit
	dog

bird
  chicken
  parrot
	crow^20

So now when we write \[animal\] the bird "sub-list" will be chosen with a probability that makes it so crow actually has a 20x chance of occurring - like it would if we actually merged the two lists by cutting and pasting all the items of the mammal and bird lists into the animal list.

Enjoy!
