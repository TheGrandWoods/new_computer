Add customEmojis = \{import:huge-emoji-list\} under your comments-plugin options list to get 80k+ emojis. Like this:

commentsOptions
  customEmojis = \{import:huge-emoji-list\}

If you want to add this list, but also include your own emojis as well, then you can do that with the comments plugin's @import feature like this:

commentsOptions
  customEmojis
    @import = \{import:huge-emoji-list\}
    cat_jam = https://user.uploads.dev/file/a43d0b52d94c91dddb00cf157dd8c989.webp
    crythumbsup = https://user.uploads.dev/file/14c78e8fbd9767e69a6b86d26817bacb.png
    ...

Note that you can only have one @import line in your customEmojis list.
