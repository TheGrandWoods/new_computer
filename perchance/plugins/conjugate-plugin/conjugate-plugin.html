Conjugate Plugin

Put this in your Perchance code panel:

conjugate = \{import:conjugate-plugin\}


Now you can conjugate verbs into any form you like:

\[conjugate("run").gerund\]

That would generate running.

In the above example I've put a word ("run") directly into the conjugate plugin, but normally you'd be putting in a randomly selected verb. Here's some example code to demonstrate how you'd do that: 

verb
	read
	ride
	run

output
  The future tense of \[v = verb.selectOne\] is \[conjugate(v).future\].

Here are all the possible conjugations:

\[conjugate("speak").past\]           --> spoke
\[conjugate("speak").infinitive\]     --> speak
\[conjugate("speak").gerund\]         --> speaking
\[conjugate("speak").actor\]          --> speaker
\[conjugate("speak").present\]        --> speaks
\[conjugate("speak").future\]         --> will speak

You can of course use it with lists too, and it'll randomly select an item from the list and conjugate it (just as you can do with myList.pastTense and the like): 

myListOfVerbs
	read
	ride
	run

output
  She is a great \[conjugate(myListOfVerbs).actor\].
