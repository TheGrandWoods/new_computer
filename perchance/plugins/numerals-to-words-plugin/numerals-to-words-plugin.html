Numerals to Words Plugin

Put this in your Perchance code panel:

numToWord = \{import:numerals-to-words-plugin\}


Now you can generate the "spelled out" version of a number:

\[numToWord(745)\]

That would generate seven hundred and forty-five.

If you don't want the "and", you can specify US-style grammar:

\[numToWord(745, "US")\] => seven hundred forty-five


You can of course use it with variables and other plugins:

\[numToWord(num)\]

\[numToWord( dice("2d6") )\]

\[numToWord( character.hitpoints )\]


Test out this plugin by typing in a number:

\[numToWord()\]

[$output(userval.value)]
