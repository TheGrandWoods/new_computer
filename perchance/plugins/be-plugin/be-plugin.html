Be Plugin

This plugin is like \{a\}, except instead of automatically deciding on a/an, it automatically decides on is/are. In other words, it automatically decides on the correct "to be" verb based on the preceding text (which is usually a pronoun in this context).

Here's a simple example of how to use it:

be = \{import:be-plugin\}

they = \{they|she|he\}

output
  I think \[they\] \[be()\] having fun.

You can use \[be().titleCase\] to get Is/Are and \[be().upperCase\] to get IS/ARE.

You can use \[be("past")\] to get was/were.

The default fallback verbs are is/was (in case the preceding word is not they/she/he/I/we).
