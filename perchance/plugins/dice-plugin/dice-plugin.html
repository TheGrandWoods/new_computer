🎲︎ Dice Plugin 🎲︎

Put this in your Perchance code panel:

dice = \{import:dice-plugin\}


Now you can use dice:

\[dice("3d12")\]


Add and subtract:

\[dice("3d12+4")\]  \[dice("2d6+1d8")\]  \[dice("4d6-3d4+2")\]


Multiply and combine with variables:

\[dice("3d12") * dice("1d20") - 50 + characterAge\]


Roll a number of dice based on a variable:

\[dice(numDice+"d6")\]


Roll a dice size based on a variable:

\[dice("2d"+diceSize)\]


Dynamic number and size:

\[dice(numDice+"d"+diceSize)\]


Name it whatever you want:

roll = \{import:dice-plugin\}

\[roll("1d6")\]


Notes:

  • "3d12" means "roll three 12-sided dice" - read about dice notation
  • Check out more plugins at perchance.org/plugins
