Typewriter Plugin

[$output("Typewriter Plugin", 4, 1)]

Put this in your Perchance code panel:

type = \{import:typewriter-plugin\}


Now put this anywhere:

\[type("My example text!")\]


You can alter the speed (characters/second) and the delay before it starts (seconds):

\[type(myList, 5, 3)\]


So the format is:

\[type(text, speed, delay)\]


Remember, you can name plugins whatever you want:

typewriter = \{import:typewriter-plugin\}

\[typewriter(coolSentence, 2, 1.5)\]



Notes:

  • This plugin is experimental - it may be somewhat buggy
  • You cannot use line breaks (<br>) within the text to be typed
  • Here's an example of how you could use it.
