Wheel Plugin

This plugin allows you to make a spinning wheel with cusomtizable segments. To use this plugin, you'll first need to import it by pasting this line in your Perchance lists editor panel: 

makeWheel = \{import:wheel-plugin\}

Now that you've imported your plugin, you'll need to define the segments for your wheel like this: 

wheelOptions
  segments
    Prize One
    Prize Two
    Prize Three
    Prize Four
    Prize Five

And now just paste this: \[makeWheel(wheelOptions)\] anywhere to make a wheel. You can paste it in your HTML panel, or as part of a list in your Perchance lists panel. It'll output this:

[$output(wheelOptions1)]

Click the wheel to make it spin!

The wheel segments are grey by default, but you can set a color for each segment like this: 

wheelOptions
  segments
    Prize One
      fillStyle = #eae56f // get color codes here: google.com/search?q=hex+color+picker
    Prize Two
      fillStyle = #89f26e
    Prize Three
      fillStyle = #7de6ef
    Prize Four
      fillStyle = #e7706f

[$output(wheelOptions2)]

As mentioned in the code above, you'll need to use a color picker to get the code for the color that you desire. There are some "named" colors that you can use instead of the "hex codes" that you see above. For example, you can write fillStyle=blue to make a segment blue.

You can use random colors like this:

wheelOptions
  randomFillStyle = pastel // or "normal"
  segments
    Prize One
    Prize Two
    ...

[$output(wheelOptions3)]

You can define the approximate number of spins that the wheel is to do, the duration of the spinning, and the spin direction like so: 

wheelOptions
  animation
    duration = 15
    spins = 20
    direction = anti-clockwise // default is "clockwise"
  segments
    Prize One
    Prize Two
    Prize Three

[$output(wheelOptions4)]

You can change the default fill style of the whole wheel like this:

wheelOptions
  fillStyle = #e7706f
  segments
    Prize One
    Prize Two
    Prize Three

[$output(wheelOptions5)]

You can change the line width like this:

wheelOptions
  lineWidth = 10
  segments
    Prize One
    Prize Two
    Prize Three

[$output(wheelOptions6)]

You can change the line color with strokeStyle:

wheelOptions
  lineWidth = 5
  strokeStyle = #e7706f
  segments
    Prize One
    Prize Two
    Prize Three

[$output(wheelOptions7)]

You can use images for each segment like this:

wheelOptions
  segments
    Apples
      image = https://user-uploads.perchance.org/file/47592ece70be9cfbc378b3820d93eae6.jpeg
    Bananas
      image = https://user-uploads.perchance.org/file/1b7bfdc0ef9998b869301f83974feacd.jpeg
    Grapes
      image = https://user-uploads.perchance.org/file/9bf235dcce14dc85e26bff744338697b.jpeg

[$output(wheelOptions8)]

Important: You should upload your images to perchance.org/upload rather than linking directly to images that you find on the internet (e.g. pinterest, discord, wikis, etc). Just find an image online, copy its URL, and paste it on the upload page (or paste the image itself, or upload an image from your computer). Once the image has uploaded, get the direct URL (ending in something like .png or .jpg or .webp, etc). This is important because the average lifespan of a URL on the internet is only a few years - so if you use an image URL from somewhere on the internet, then your generator will likely break after a few years. The Perchance uploader keeps your images online forever.

You'll also want to make sure that your image isn't tooo large. You can use photopea.com to resize your images. Just open the image and then click "File > Export As > JPG" and you can change the size and compression quality. Ideally the sum of all your images sizes should be under 5MB if you want your generator to load quickly.

Also, the images don't render in the correct place if their segment's angle/size is larger than 180 degrees. I haven't yet gotten around to fixing this - sorry! For now you'll just have to make sure that no segment takes up more than half the wheel.

You can change the "tick" sound that is made by the wheel as it spins like this: 

wheelOptions
  sound = tick // or tock or tock2 or none or data:audio/mpeg;base64,.....
  segments
    Prize One
    Prize Two
    Prize Three

The three presets are "tick", "tock" and "tock2". The "tock" sound is default. If you'd like to use your own custom sound, then find a short audio clip (preferably an mp3 file), download it to your computer, and then upload it using the file picker below. It should be a very short audio clip. If the audio file is too large (e.g. larger than 20kb), then it may be laggy when you try to use it.

Once you've done that, a text box should appear with text in it starting with something like data:audio/mpeg;base64,...... Copy this text (all of it) and paste it as the sound option like this: 

wheelOptions
  sound = data:audio/mpeg;base64,.....
  segments
    Prize One
    Prize Two
    Prize Three

And now your custom sound will play as the wheel spins instead of the default "tock" sound.

You can change the segment sizes by setting the segment size properties to values that add up to 360 degrees. Here's an example: 

wheelOptions
  segments
    Prize One
      size = 180
    Prize Two
      size = 90
    Prize Three
      size = 90

[$output(wheelOptions11)]

There are all sorts of text options that you can use to customize the display of the text: 

wheelOptions
  textFontSize = 25
  textOrientation = curved
  textAlignment = inner
  textMargin = 130
  textFontFamily = monospace
  textStrokeStyle = black
  textLineWidth = 3
  textFillStyle = white
  segments
    Prize One
    Prize Two
    Prize Three

[$output(wheelOptions9)]

You can change the pointer icon and wheel diameter like this:

wheelOptions
  pointer = ⬇️
  wheelDiameter = 300
  segments
    Prize One
    Prize Two
    Prize Three

[$output(wheelOptions12)]

Here's the detault pointer:

wheelOptions
  pointer = <div style="font-size:200%; transform: translateY(10px);">▼</div>
  ...

You can change its color like this:

wheelOptions
  pointer = <div style="color:#e7706f; font-size:200%; transform: translateY(10px);">▼</div>
  ...

(Change #e7706f to a color code of your choice)

You can run some Perchance/JavaScript code that depends on the result of the wheel spin like this: 

wheelOptions
  animation
    callbackFinished(winningSegment) => alert("You won "+winningSegment.text)
  segments
    Prize One
    Prize Two
    Prize Three

[$output(wheelOptions10)]

This is useful if you want to generate some text based on the result of the spin. Here's an example generator that puts the winning text into a variable and then randomizes the generator.
