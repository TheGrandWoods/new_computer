Title Case Plugin

This plugin is a "smarter" version of the built-in .titleCase. For example, it doesn't capitalize "a", "an", "the" and other words as defined here. You can use it like this: 

titleCase = \{import:title-case-plugin\}

output
  INPUT: \[t = titles.selectOne\] <br> OUTPUT: \[titleCase(t)\]

titles
  this is an example          // → This Is an Example
  an example headline         // → An Example Headline
  this is the third example   // → This Is the Third Example
  the fourth and last example // → The Fourth and Last Example
