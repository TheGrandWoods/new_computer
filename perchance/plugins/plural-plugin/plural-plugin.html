Plural Plugin

If you want to define plurals for words that pluralForm doesn't cover, then this is the plugin for you. To use this plugin, instead of writing this: 

output
  There are a few \[animal.pluralForm\] over there.

you write this:

plural = \{import:plural-plugin\}

output
  There are a few \[plural(animal)\] over there.

To add your own pluralization rules, add a list like this to your Perchance lists editor: 

pluralRules
  bobbin :: bobbinses      // bobbinses is plural of bobbin
  zib :: zibz              // zibz is plural of zib
  *gex :: *gexii           // plural of words ending in gex is *gexii
  furniture :: furniture   // don't add an s to the end (it's an "uncountable" noun)

And then paste this at the top of your HTML editor (the bottom-right panel): 

\[plural(pluralRules, "ADD_RULES")\]

Important note: When you add or remove rules, you need to click the reload button to refresh your generator and have the changes go into effect.

You can get the singular form like so:

output
  There are a few \[plural(animal, 1)\] over there.

And if you put any number after it, and then true after that, like this: 

output
  There are \[plural(animal, 7, true)\] over there.

Then it will output something like this:

There are 7 piglets over there.

In other words, it adds the number before the word, and then pluralizes the word based on the number. This is handy for cases like this where you don't know whether the word should be pluralized or not because the number is randomly generated: 

num = {1-10}

output
  \[n = num.selectOne, ""\] There are \[plural(animal, n, true)\] over there.
